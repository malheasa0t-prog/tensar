# برومبت تصميم صفحة تسجيل دخول تعليمية (Login Page Blueprint Prompt)

**الهدف من البرومبت:** 
مساعدة الذكاء الاصطناعي على قراءة صور صفحة تسجيل الدخول الخاصة بك وتحويلها إلى كود HTML و CSS تعليمي، بحيث يحتوي على إطارات توضيحية حول العناصر مع شرح احترافي يوفر تجربة تعليمية تشبه "المخططات الهندسية" (Blueprints).

---

## انسخ البرومبت التالي:

```text
الدور (Role): أنت مهندس واجهات أمامية (Front-end Engineer) وخبير في تجربة واجهة المستخدم (UI/UX) ولديك مهارة عالية في تبسيط وشرح التصاميم لتحويلها إلى أكواد تعليمية.

المهمة (Task): سأقوم بإرفاق صور لصفحة "تسجيل الدخول" (Login Page) الخاصة بموقعي. أطلب منك تحويل هذا التصميم بدقة إلى كود HTML و CSS حديث واحترافي.

المتطلبات التقنية والتعليمية:
1. التأطير البصري (Highlights & Frames): قم بإضافة إطارات (Borders) توضيحية أنيقة (مثلاً بخطوط متقطعة أو ألوان بارزة) حول كل عنصر أساسي في الصفحة (مثل: حقل الإيميل، حقل كلمة المرور، زر الدخول، الشعار، نسيان كلمة المرور).
2. الشرح المدمج (UI Annotations): أضف نصوصاً توضيحية (Labels) أو بطاقات صغيرة بجانب كل عنصر مشمول بالإطار. يجب أن يشرح النص وظيفة الحقل برمجياً، وكيف يساهم في تحسين تجربة المستخدم (UX)، والسبب وراء تصميمه بهذا الشكل بناءً على أفضل الممارسات.
3. تعليقات الكود (Code Comments): اكتب تعليقات تفصيلية واحترافية داخل كود الـ HTML والـ CSS تشرح وظيفة كل حاوية (Container) وكل تنسيق (Style) تم استخدامه.
4. التجاوب (Responsiveness): يجب أن يكون الكود متجاوباً بالكامل ليعمل بشكل سليم على الجوالات والأجهزة اللوحية والشاشات الكبيرة.
5. البيئة التعليمية (Blueprint Style): أريد أن تبدو النتيجة النهائية في المتصفح وكأنها "مخطط تعليمي" (Interactive Blueprint) يشرح تشريح (Anatomy) صفحة تسجيل الدخول الخاصة بي بشكل احترافي ومنسق.
6. الكود النهائي: يرجى كتابة كود نظيف وسهل القراءة (Clean Code) وتزويدي بالكود كاملاً جاهزاً للتشغيل في المتصفح.
```

---

## طريقة الاستخدام:
1. افتح نموذج الذكاء الاصطناعي الذي يدعم رفع الصور (مثل ChatGPT-4o، أو Claude 3.5 Sonnet، أو Gemini 1.5 Pro).
2. قم برفع صور صفحة تسجيل الدخول الخاصة بك.
3. الصق البرومبت بالكامل باللون الرمادي أعلاه في صندوق المحادثة.
4. سيقوم الذكاء الاصطناعي بتوليد صفحة تفاعلية مذهلة تشرح للمطورين كيف تم بناء واجهتك.
